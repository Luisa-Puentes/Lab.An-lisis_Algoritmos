# Laboratorio de Análisis de Algoritmos - ITM
Este repositorio contiene la implementación en Python de algoritmos basados en el paradigma **Divide y Vencerás**.

## Contenido
- **Punto 2.4:** Comparación entre Fuerza Bruta y Divide y Vencerás para el problema del subarreglo máximo.
- **Punto 2.5:** Comparación experimental entre Merge Sort e Insertion Sort.

## Tecnologías usadas
- Python 3.12
- Matplotlib & Seaborn para visualización de datos.
